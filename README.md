# mssqlClient

Auto-generated client artifacts for MSSQL ORM. Contains TypeScript, Python, and .NET model types, metadata, and entity classes generated from `.mssql` schema files.

## Structure

```
mssqlClient/
├── typescript/          # TypeScript types and metadata
│   ├── index.ts         # Barrel exports
│   ├── base.ts          # Base types, filters, error classes
│   ├── mssqlMetadata.ts # Model-table mappings, relations
│   ├── User.ts          # Generated User model
│   └── Order.ts         # Generated Order model
├── python/              # Python metadata
│   └── mssql_metadata.py
└── dotnet/              # .NET entity classes
    ├── User.cs
    ├── Order.cs
    ├── MssqlDbContext.cs

```

## Usage

### TypeScript

```typescript
import { User, Order, Mssql } from './mssqlClient/typescript';
import { modelToTable, relationMap } from './mssqlClient/typescript/mssqlMetadata';

// Use generated types
const user: User = { id: '123', email: 'john@example.com', name: 'John', createdAt: new Date() };

// Access metadata
console.log(modelToTable['user']); // '[dbo].[users]'
console.log(relationMap['order']); // { user: { ... } }
```

### Python

```python
from mssqlClient.python.mssql_metadata import MODEL_TO_TABLE, MODEL_FIELDS, RELATION_MAP

print(MODEL_TO_TABLE['user'])  # '[dbo].[users]'
print(MODEL_FIELDS['user'])    # {'id': 'string', 'email': 'string', ...}
```

### .NET

```csharp
using MssqlOrm.Entities;

var user = new User { Id = "123", Email = "john@example.com" };
var db = new MssqlDbContext(connectionString);
var users = db.Users.FindMany();
```
