# an5Client

Auto-generated client artifacts for AN5 ORM. Contains TypeScript, Python, and .NET model types, metadata, and entity classes generated from `.an5` schema files.

## Structure

```
an5Client/
├── typescript/          # TypeScript types and metadata
│   ├── index.ts         # Barrel exports
│   ├── base.ts          # Base types, filters, error classes
│   ├── an5Metadata.ts # Model-table mappings, relations
│   ├── User.ts          # Generated User model
│   └── Order.ts         # Generated Order model
├── python/              # Python metadata
│   └── an5_metadata.py
└── dotnet/              # .NET entity classes
    ├── User.cs
    ├── Order.cs
    ├── An5DbContext.cs
    └── An5Config.cs
```

## Usage

### TypeScript

```typescript
import { User, Order, An5 } from './an5Client/typescript';
import { modelToTable, relationMap } from './an5Client/typescript/an5Metadata';

// Use generated types
const user: User = { id: '123', email: 'john@example.com', name: 'John', createdAt: new Date() };

// Access metadata
console.log(modelToTable['user']); // '[dbo].[users]'
console.log(relationMap['order']); // { user: { ... } }
```

### Python

```python
from an5Client.python.an5_metadata import MODEL_TO_TABLE, MODEL_FIELDS, RELATION_MAP

print(MODEL_TO_TABLE['user'])  # '[dbo].[users]'
print(MODEL_FIELDS['user'])    # {'id': 'string', 'email': 'string', ...}
```

### .NET

```csharp
using An5Orm.Entities;

var user = new User { Id = "123", Email = "john@example.com" };
var db = new An5DbContext(connectionString);
var users = db.Users.FindMany();
```
